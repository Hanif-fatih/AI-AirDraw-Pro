# 📌 Overview

AI Air Drawing Studio Pro is an advanced computer vision application that transforms your hand into a virtual pen.

Using Google's **MediaPipe Hand Landmarker**, the system tracks hand movements in real time and allows users to draw in the air without touching any physical device.

Unlike simple air drawing demos, this project focuses on:

- Professional desktop interface
- Low-latency rendering
- High FPS performance
- AI-based gesture recognition
- Modern brush engine
- Beautiful visual effects
- Optimized architecture
- Production-quality code organization

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Air-Drawing-Studio-Pro.git

cd AI-Air-Drawing-Studio-Pro
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python src/app.py
```
